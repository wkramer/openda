Costa documentation

User Documentation

  • COSTA Getting Started document - presents an overview of the structure of the COSTA-system.
  • COSTA Reference manual - more detailed information on the usage of various COSTA components.
  • COSTA course material - this provides the course material developed in 2007/8.
  • COSTA on-line Wiki system - the wiki on the COSTA website provides additional information on data-assimilation and the usage of the COSTA system.

Programmer Documentation

  • COSTA Developers Guide - this provides an overview of the directory structure, installation and compilation procedures, and testing of the COSTA system.
  • COSTA Doxygen documentation - the Doxygen documentation is generated from the code and provides the complete headers of the routines.
  • COSTA memos, design documents - overview of all COSTA memos, references to design documents
